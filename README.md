# 🏆 Taitaja - Frågesportsapplikation  

Detta är en webbaserad frågesportsapplikation utvecklad för Taitaja-tävlingen. Applikationen låter användare välja en lärare, en kategori och antalet frågor innan spelet börjar.  

## 🚀 Funktioner  
- Välj en **lärare** från en dropdown-meny.  
- Välj en **kategori** från en dropdown-meny.  
- Välj antalet frågor via radioknappar.  

## 🛠 Teknologi  
- **React.js** (Next.js) – Frontend-ramverk  
- **Axios** – API-anrop  
- **CSS** – Styling (utan Tailwind)
